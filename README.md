# FinanceFlow

FinanceFlow is a full-stack personal finance management application built to help users track their income, expenses, savings, and financial goals in one place.

The application allows users to create an account, securely log in, manage transactions, set savings goals, and view their financial data through interactive charts and a clean dashboard. All user data is stored in MongoDB, ensuring each user has their own private workspace.

## Features

* User authentication (Sign Up & Login)
* Dashboard with financial overview
* Add, edit and delete transactions
* Create and manage savings goals
* Interactive analytics and charts
* Profile and settings management
* Secure data storage using MongoDB

## Tech Stack

* React
* TypeScript
* Express.js
* MongoDB Atlas
* JWT Authentication
* CSS

## Getting Started

```bash
npm install
npm run dev
```

Create a `.env` file with your MongoDB connection string and JWT secret before running the project.
